# HTML Demos Index

This directory contains various HTML demonstration and test files for the trading bot project.

## Main Pages
- `index.html` - Main application index page
- `dashboard.html` - Trading dashboard demo
- `backtest.html` - Backtest results demo

## Test Pages
- `test_chart.html` - Chart.js integration tests
- `test_chart_integration.html` - Chart integration testing
- `test_dark_theme.html` - Dark theme functionality test
- `test_theme_complete.html` - Complete theme system test
- `test_theme_toggle.html` - Theme toggle functionality
- `test_equity_curve.html` - Equity curve display test
- `test_auto_refresh.html` - Automatic chart refresh test

## Usage
Open any HTML file in a web browser to view the demo or test:
```bash
# Example
open test_chart.html
```

## Dependencies
Most test files require:
- Chart.js (loaded from CDN)
- Modern web browser with JavaScript enabled
- Some files may require the backend API to be running

## Purpose
These files were used during development to test individual components and features before integration into the main Vue.js application.
