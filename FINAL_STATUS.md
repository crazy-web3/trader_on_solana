## 🔗 API 端点

### 基础端点
```
GET  /api/health          - 健康检查
GET  /api/symbols         - 获取支持的币种
GET  /api/intervals       - 获取支持的时间周期
```

### 数据端点
```
GET  /api/klines          - 获取K线数据
     参数: symbol, interval, days
```

### 缓存端点
```
GET  /api/cache/stats     - 获取缓存统计
POST /api/cache/clear     - 清空缓存
```
