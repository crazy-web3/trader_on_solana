## 🎯 快速开始

### 1️⃣ 打开前端页面
```
直接打开 index.html 文件
```

### 2️⃣ 查询数据
- 选择币种: BTC/USDT, ETH/USDT, BNB/USDT, SOL/USDT
- 选择周期: 1m, 5m, 15m, 1h, 4h, 1d, 1w
- 设置天数: 1-365天
- 点击查询

### 3️⃣ 查看蜡烛图
- 🟢 **绿色蜡烛** = 上涨
- 🔴 **红色蜡烛** = 下跌

## 📊 蜡烛图说明

```
    ┌─────────────┐
    │   最高价    │
    ├─────────────┤
    │             │
    │  实体部分   │  绿色=上涨，红色=下跌
    │             │
    ├─────────────┤
    │   最低价    │
    └─────────────┘
```
